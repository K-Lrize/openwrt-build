# Global package pool for the SDK build.
#
# 这里写的是「裸包名」（不是源码路径）。pool 轨把这些包名翻译为
# CONFIG_PACKAGE_<name>=m，喂给 SDK 容器里的 make defconfig，然后一次
# make -j package/compile —— buildroot 自己按依赖图调度，省得我们手卷路径表。
#
# 规则：
#   - 一行一个包名；支持整行 # 注释与行尾 # 注释；空行 OK。
#   - 字符集 [a-zA-Z0-9._+-]+；包名重复 / 含空格会被 lint 拦下。
#   - 禁止 kmod-*：SDK 的内核 .config 在完整 buildroot 阶段已固化，事后无法编出
#     base-config 之外的内核符号。需要的 kmod 请在 common/base-config 里用
#     CONFIG_PACKAGE_kmod-xxx=m 声明，由 base.yml 的完整 buildroot 流程产出。
#   - 拼写错或 feed 里不存在的包名会在 SDK 容器 make defconfig 之后被列出来：
#     pool 轨 strict 模式直接红，Tier3 lenient 模式 warning 继续。
#
# 静态校验入口：scripts/ci/lint-custom-packages.sh（被 validate.yml 调用）。
